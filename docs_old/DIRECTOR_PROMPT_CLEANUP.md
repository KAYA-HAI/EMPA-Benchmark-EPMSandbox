# Director Prompt 清理说明

## 📝 清理内容

### 1️⃣ 删除废弃的 Progress 分数模式

**之前**: 支持两种模式
- 旧模式：单一 progress 分数（0-100）
- 新模式：EPJ/EPM 向量系统

**现在**: 仅支持 EPJ/EPM 向量系统
- 所有对话都使用 EPJ 向量（P_0, P_t, v_t）
- 启用时自动包含 EPM v2.0 能量动力学分析

### 2️⃣ 简化函数签名

**之前**:
```python
def generate_director_prompt(
    progress: int = None,  # ❌ 已废弃
    epj_state: dict = None,
    history: list = None,
    ...
)
```

**现在**:
```python
def generate_director_prompt(
    epj_state: dict,  # ✅ 必需参数，不再可选
    history: list = None,
    ...
)
```

### 3️⃣ 删除旧的决策指南

**删除的内容** (27行):
```python
# 旧模式的决策指南（基于分段）
### 对话初期（进度0-30）
- 主要策略：建立对话，观察AI的共情能力
...

### 对话中期（进度30-60）
...

### 对话深入（进度60-90）
...

### 接近结束（进度90+）
...
```

**保留的内容**:
- ✅ 基于 EPJ 向量的决策指南
- ✅ 基于 EPM 能量动力学的策略补充
- ✅ 对话失败终止条件
