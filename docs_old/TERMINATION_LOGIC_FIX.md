# Director终止逻辑修复

## 问题描述

在script_009的测试中，Director在**未满足任何EPM科学条件**的情况下就提前结束了对话：

**错误的终止时机：**
- 轮次：9/45（远未达到最大轮次）
- 累计能量：E_total = 14.58 / 28.05 (52.0%) ❌ 未达标
- 当前距离：r_t = 14.00，阈值 = 2.81 ❌ 未进入安全区
- 位置投影：projection = -13.47，阈值 = -2.81 ❌ 未穿越

**错误的终止理由：**
> "所有预设的剧情阶段都已释放，且倾诉者情绪积极饱满，适合结束对话。"

这个理由**完全违背了EPM的科学标准**！

## 根本原因

Director的prompt中混淆了剧情管理和EPM科学标准，包含了以下**错误的终止条件**：

```python
#### 🚨 必须结束对话的情况（旧版本-错误）

1. 能量胜利: E_total达到阈值 ✅ 正确
2. 几何胜利: r_t进入目标区域 ✅ 正确
3. 位置胜利: projection穿越阈值 ✅ 正确
4. 方向崩溃: 连续3步负能量，且资源已耗尽 ❌ 错误
5. 资源耗尽: 剧情全释放，记忆>2个，r_t仍高 ❌ 错误
6. 对话循环: 重复表达且调整无效 ❌ 错误
```

LLM被第5条"资源耗尽"误导，认为"剧情全释放"是可以结束的理由。

## 修复方案

### 1. 修正Director Prompt (`director_prompts.py`)

**新的严格规则：**

```python
#### 🎯 EPM胜利条件（任一满足即可结束对话）

1. **几何胜利**: r_t ≤ ε_distance （距离进入目标区域）
2. **位置胜利**: projection ≥ -ε_direction （成功穿越阈值）
3. **能量胜利**: E_total ≥ ε_energy （累积能量达标）

**当EPM状态包中显示任一胜利条件的✅标记时，你才可以调用end_conversation结束对话。**

#### 🚨 终止对话的严格规则

**只能在以下情况结束对话：**

1. ✅ **EPM科学胜利**：上述三个条件之一满足（会在EPM摘要中显示✅）
2. ⏱️ **超时终止**：达到最大轮次（系统会自动终止）

**绝对禁止的错误理由：**
- ❌ "剧情阶段都释放了" —— 剧情释放≠共情成功
- ❌ "情绪积极饱满" —— 主观判断≠科学指标
- ❌ "对话达到自然终点" —— 叙事完整≠EPM达标
- ❌ "资源耗尽" —— 剧情只是工具，不是终止条件

**关键原则：**
- 剧情阶段的释放状态**与终止决策无关**
- 释放完所有剧情后，**仍应继续对话**，直到EPM条件满足
- 即使对话"感觉很完美"，**也必须等待EPM指标达标**
- 你的任务是**帮助TestModel达成EPM胜利**，而不是讲完一个故事
```

### 2. 修正函数描述 (`director_function_schemas_selector.py`)

**新的end_conversation描述：**

```python
{
    "name": "end_conversation",
    "description": """结束对话。**只能在EPM科学胜利条件达成时调用**。

🚨 严格要求：
- 只有当EPM状态包显示任一胜利条件的✅标记时才能调用此函数
- 三个胜利条件：几何胜利(r_t ≤ ε_distance)、位置胜利(projection ≥ -ε_direction)、能量胜利(E_total ≥ ε_energy)

❌ 绝对禁止的错误调用：
- "剧情阶段都释放了" —— 这不是终止条件
- "倾诉者情绪充分恢复" —— 主观判断不可靠
- "对话达到自然终点" —— 叙事完整≠EPM达标

你的任务是帮助TestModel达成EPM胜利，而不是讲完一个故事。"""
}
```

### 3. 修正决策指南中的描述

**修改前（容易误导）：**
- "持续上升 → AI共情失败，需要强力干预或考虑结束"
- "方向崩溃预警 → 立即释放强力素材或果断结束"

**修改后（明确只能干预）：**
- "持续上升且高 → AI共情失败，需要强力干预（释放强力素材/调整策略）"
- "崩溃预警 → 立即释放强力素材，尝试扭转局面"

## EPM终止条件科学标准

### 有效的终止条件（仅2种）

| 条件 | 判断标准 | 说明 |
|------|---------|------|
| **几何胜利** | `r_t ≤ ε_distance` | 当前位置距原点足够近 |
| **位置胜利** | `projection ≥ -ε_direction` | 成功穿越目标区域（从负半空间到正半空间） |
| **能量胜利** | `E_total ≥ ε_energy` | 累积有效共情能量达到初始赤字水平 |
| **超时终止** | `turn_count >= max_turns` | 达到最大轮次（系统自动终止，非Director决策） |

### 无效的终止理由（全部删除）

| ❌ 错误理由 | 为什么错误 |
|-----------|----------|
| "剧情阶段都释放了" | 剧情只是工具，释放状态与共情成功无关 |
| "情绪积极饱满" | 主观判断不可靠，必须用科学指标 |
| "对话达到自然终点" | 叙事完整性≠EPM目标达成 |
| "资源耗尽" | 剧情耗尽后仍可用记忆、策略继续推进 |
| "对话循环/重复" | 应该通过释放新素材打破循环，而非放弃 |

## 验证方法

修复后，Director应该：
1. ✅ 在剧情全部释放后**继续对话**，直到EPM达标
2. ✅ 只在EPM摘要显示`✅`标记时才结束对话
3. ✅ 结束理由必须明确说明满足了哪个EPM胜利条件
4. ❌ 绝不以"剧情完整"、"情绪良好"等主观理由结束

## 重要性

这个修复**至关重要**，因为：

1. **科学严谨性**：EPM是一个科学评测系统，必须严格遵循数学定义的终止条件
2. **评测公平性**：如果允许主观判断终止，不同模型的测试轮次会不一致，失去可比性
3. **指标准确性**：提前终止会导致EPM指标无法真实反映模型的共情能力
4. **系统可信度**：作为benchmark系统，终止逻辑的随意性会严重影响结果的可信度

## 总结

**核心原则：**
> Director的唯一任务是**帮助TestModel达成EPM胜利**，而不是讲完一个故事。
> 剧情、记忆都只是工具，EPM科学指标才是唯一的成功标准。

**修复后的行为：**
- ✅ 严格按照EPM三个胜利条件判断是否结束
- ✅ 剧情释放状态完全不影响终止决策
- ✅ 在EPM达标前持续寻找推进方法
- ✅ 只有科学指标达标或超时才终止对话
